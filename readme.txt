== Description ==

Puts Facebook Like and Share buttons of your choice above or below your posts.
Easy customization of active buttons and position in the Settings menu.

 Screenshots :

http://wp-plugins-themes.com/wp-plugin/facebook-like-and-share-button-wp-plugin/

== Installation ==
1. Upload the `facebook-like-share-button` directory into the `/wp-content/plugins/` directory
2. Set your favourite values in the `Settings->facebook like share` menu in Wordpress
3. Activate the plugin through the `Plugins` menu in WordPress
4. Enjoy!
